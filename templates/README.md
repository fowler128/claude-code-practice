# Templates Directory

Comprehensive business, job search, and personal infrastructure templates.

## Directory Structure

```
templates/
├── bizdeedz/           # BizDeedz consulting business templates
├── job-search/         # Career transition and job hunt tools
└── personal/           # Personal and family infrastructure
```

## Usage Notes

### Markdown Files (.md)
- Open in any text editor or Notion
- Copy/paste into Word for formatting
- Use as-is for digital delivery

### CSV Files (.csv)
- Open directly in Excel or Google Sheets
- All formulas are text-based for reference
- Add your own formulas for calculations

## Quick Start

1. **BizDeedz**: Start with `client-proposal-template.md` and `statement-of-work-template.md`
2. **Job Search**: Begin with `application-tracking-dashboard.csv` to systematize your search
3. **Personal**: Complete `emergency-fund-runway-calculator.csv` for financial clarity
