# BizDeedz AI Readiness Offer Skill

Standardizes the BizDeedz AI Readiness Assessment into a consistent, premium, repeatable offer — close in 20–30 minutes, zero freelancing.

## Triggers

Use this skill when the user asks about:
- AI readiness, assessment scope, or deliverables
- Pricing or proposal language for the assessment
- Sales call scripts or discovery questions
- Objection handling
- Statement of Work or engagement kick-off

## Offer At A Glance

| | |
|---|---|
| **Name** | AI Readiness Assessment (Scorecard + Stabilization Roadmap) |
| **Timeline** | 5–7 business days from kickoff |
| **Focus** | 1–2 workflows max (no scope creep) |
| **Phase 2 bridge** | Hands off to `bizdeedz-workflow` for implementation |

## Deliverables (Every Engagement)

1. Readiness Scorecard (7 dimensions)
2. Current-state workflow map
3. Pain heatmap + risk register
4. Prioritized "do this first" backlog
5. 30/60/90 stabilization roadmap
6. Automation opportunity brief
7. Governance starter kit (SLA, escalation, RACI, audit logging)

## Pricing Tiers

| Tier | Scope | Range |
|------|-------|-------|
| A — Assessment (Core) | 1 workflow | $2,500–$5,000 |
| B — Assessment Plus | 2 workflows + governance kit + KPI spec | $5,000–$8,500 |
| C — Assessment + Phase 2 Credit | Tier B + implementation credit | $7,500–$12,500 |

## Output Formats

- **A** — One-page offer sheet
- **B** — Proposal scope + tier recommendation
- **C** — Sales call script + discovery questions
- **D** — Statement of Work outline
- **E** — Delivery plan + checklist (5–7 day)
