# Claude Code Skills

This directory contains integrated Claude Code skills that extend the functionality of the Claude Code development environment.

## Available Skills

### Humanizer

A Claude Code skill designed to identify and remove AI-generated writing patterns from text, making it sound more natural and human-written.

**Location:** `./humanizer/`

**Features:**
- Detects 24 distinct patterns associated with AI writing
- Identifies content issues, language quirks, and style problems
- Performs multi-pass analysis for comprehensive AI-pattern removal
- Based on Wikipedia's "Signs of AI writing" guide

**Files:**
- `SKILL.md` - The skill definition and implementation
- `README.md` - Detailed documentation and usage examples
- `WARP.md` - Additional implementation notes

**Usage:**
Refer to the humanizer skill's README.md for complete usage instructions and examples.

## Integration Notes

These skills are integrated directly into the Claude Code practice project and can be used by invoking them through the Claude Code interface.
