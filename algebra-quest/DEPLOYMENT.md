# Deployment Guide (Vercel)

This app is configured to deploy on Vercel as a Vite frontend with a serverless API endpoint at `api/generate.js`.

## 1) Prerequisites

- Vercel account
- Anthropic API key
- This repository connected to GitHub

## 2) Import project to Vercel

1. In Vercel, click **Add New Project** and import this repo.
2. Set the **Root Directory** to `algebra-quest`.
3. Build settings should auto-detect from `vercel.json`:
   - Build Command: `npm run build`
   - Output Directory: `dist`

## 3) Configure environment variables

In Vercel project settings, add:

- `ANTHROPIC_API_KEY` = your Anthropic API key

## 4) First deploy

- Click **Deploy** in Vercel, or push to `main`.
- The app URL will be generated by Vercel.

## 5) Optional CI/CD automation

A GitHub Actions workflow is included at `.github/workflows/deploy-vercel.yml`.
To enable automatic production deployment on pushes to `main`, configure these repository secrets:

- `VERCEL_TOKEN`
- `VERCEL_ORG_ID`
- `VERCEL_PROJECT_ID`

## 6) Verify deployment

After deploy:

1. Load the app home page.
2. Start a lesson and generate a practice problem.
3. Confirm the API route works by ensuring no `ANTHROPIC_API_KEY not configured` errors appear.

## Troubleshooting

- **500: `ANTHROPIC_API_KEY not configured`**: missing env var in Vercel.
- **Build fails**: run `npm ci && npm run build` inside `algebra-quest` locally.
- **API failures**: verify Anthropic key is valid and has quota.
