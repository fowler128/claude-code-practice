BizDeedz Mission Control

This repository now targets a strict Phase 1 foundation: Next.js app shell, protected routes, Supabase auth wiring, Tailwind + shadcn/ui styling primitives, and Prisma connection setup.

Run the application with npm run dev and use the App Router pages under /dashboard, /leads, /deals, /clients, /engagements, /tasks, /finance, /knowledge, and /settings.
